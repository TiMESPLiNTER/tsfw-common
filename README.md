tsfw-common
===========
This framework component deals with everyday operations for strings, arrays, other scalar data types and even some classes.

StringUtils
-----------
* afterFirst
* afterLast
* between
* beforeFirst
* beforeLast
* endsWith
* insertBefore
* insertAfter
* startsWith
* tokenize
* urlify

ArrayUtils
----------
* arrayColumn

JsonUtils
---------
* decode
* decodeFile
* minify
